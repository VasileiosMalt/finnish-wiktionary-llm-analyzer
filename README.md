## Overview

**Finnish Wiktionary + LLM Verb Analyzer** is a Chrome extension that helps you analyze Finnish words instantly. It retrieves verb forms and translations from Wiktionary, and can optionally use advanced language models (LLMs) for fallback analysis if Wiktionary data is missing.

## Features

- Analyze Finnish verbs and words directly from your browser
- Context menu integration: right-click any word to analyze
- English-to-Finnish translation support
- Optional fallback to AI language models for enhanced results
- User-friendly popup interface

## Installation

Get the extension from the [Chrome Web Store](https://chromewebstore.google.com/detail/finnish-wiktionary-+-llm/febiphimcjnalbijoknboacgijgbocba).

## Privacy

- No personal data is collected.
- All processing is performed locally or via user-configured APIs.
- No remote code is executed.

## Permissions

- **contextMenus**: For right-click analysis of selected text
- **storage**: To save user preferences and API keys
- **Host permissions**: To access Wiktionary and supported AI APIs
